## ⚙️ Funcionalidades Implementadas
* **Autenticación de Usuarios:** Sistema de Login estructurado para el control de acceso.
* **Operaciones CRUD Completas:** Lógica implementada para Crear, Leer, Actualizar y Borrar registros en tiempo real.
* **Gestión Modular:** Interfaz dividida por categorías para la administración de Autores, Libros, Inventario, Empleados y Usuarios.
* **Visualización de Datos:** Integración dinámica de datos utilizando componentes DataGridView.

## 🛠️ Stack Tecnológico Utilizado
* **Lenguaje Principal:** C#
* **Framework de Interfaz:** Windows Forms (.NET)
* **Motor de Base de Datos:** Microsoft SQL Server
